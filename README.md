# FIKR-E-RAZA Mission

## Introduction

FIKR-E-RAZA Mission is a modern Islamic Organization Management System designed to support Dawah, education, member management, digital records, and community services.

## Key Features

- Admin Panel
- Member Panel
- PDF Library
- Attendance
- Donations
- Expense Reports
- Digital ID Cards
- Notices
- Gallery
- Contact
